# Data Analysis Agent - Frontend Interface

Это frontend-часть для автономного ИИ-агента аналитики данных. Разработан на Next.js 14+ (App Router) с использованием React, TypeScript, Tailwind CSS, Framer Motion и Recharts.

Проект содержит премиальный дизайн для демо и показа на соревнованиях или хакатонах, с глубоко интегрированными интерактивными мок-данными для демонстрации пайплайна агента.

## Окружение и Запуск

Проект использует `npm`. 

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:
```bash
npm install
```
3. Запустите в режиме разработки:
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Доступные команды

- `npm run dev` — Запуск локального dev-сервера с Hot-Reload.
- `npm run build` — Сборка продакшн-версии приложения.
- `npm run start` — Запуск собранного приложения.
- `npm run lint` — Запуск линтера ESLint.

## Кастомизация и интеграция

В данный момент frontend полностью изолирован от бекенда (Mock-only), но структура подготовлена для легкой привязки к реальному API.

**Где менять тексты и данные:**
Все текстовые метки, заголовки, списки, данные для графиков и члены команды вынесены в отдельный дата-файл.
- Откройте `src/data/site.ts`.
- В этом файле можно заменить имена, роли (массив `team`), шаги пайплайна, данные секции сравнения, и демо-превью для графиков (`demo.chartData`), а также тексты инсайтов (`demo.insights`).
- Для замены иконок: перейдите в файлы секций (например, `src/components/sections/ArchitectureSection.tsx`) и замените импорты из `lucide-react`.

**Подготовка к Backend интеграции:**
- Основной демо-интерфейс находится в `src/components/sections/ProductPreviewSection.tsx`.
- Состояние `step` управляет визуализацией. Замените `setTimeout` в `startAnalysis()` на реальные запросы через `fetch` или WebSockets для стриминга анализа.

## Дизайн

Дизайн-система основана на:
- **Цвета**: Флэт Dark Mode (Graphite: #0a0a0a, Teal/Emerald: #10b981).
- **Стили**: Glassmorphism панели, радиальные glow эффекты через Tailwind.
- **Шрифты**: Inter (встроен с помощью `next/font/google`).
- **Анимации**: Motion для React (`framer-motion`).
